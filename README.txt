Sushibot

A python bot that plays sushi-go-round. This bot can finish sushi-go-round but could use some improvements - it repeatedly makes the same plate until the customer order bubble disappears.

demo: http://youtu.be/wdKrDrGhE64

game: http://www.miniclip.com/games/sushi-go-round/en/

dependencies:
numpy - http://www.numpy.org/
PIL - http://www.pythonware.com/products/pil/
